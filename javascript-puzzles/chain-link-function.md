## Chain Link Function 

Write a function to satisfy these requirements:

```js
chain_link_function(3) // -> 3

chain_link_function(3)(4) // -> 7

chain_link_function(3)/* ... */(7) // -> a large number

chain_link_function(3)()(4) // -> 7
```
